1. Напишите программу которая генерирует все потенциальные анаграмы для 
введённой строки. (тестовое задание при устройстве на работу)
Например при вводе строки «biro» в рещультате должны получить:
biro bior brio broi boir bori ibro ibor irbo irob iobr iorb rbio rboi 
ribo riob roib robi obir obri oibr oirb orbi orib
Решение данной задачи представлено на языке C#. Исходный код в архиве Perestanovki. Выполнено в MS VS2010.

3. Конвертер арабских чисел в римские. (тестовое задание при устройстве на работу)
I = 1
V = 5
X = 10
L = 50
C = 100
D = 500
M = 1000
Например:
1) 1990 = MCMXC (M = 1000 + CM = 900 + XC = 90)
2) 2008 = MMVIII (MM = 2000 + VIII = 8)
3) 99 = XCIX (XC = 90 + IX = 9)
4) 47 = XLVII (XL = 40 + VII = 7)
Решение данной задачи представлено на языке C#. Исходный код в архиве Decimal2Rome. Выполнено в MS VS2010.
